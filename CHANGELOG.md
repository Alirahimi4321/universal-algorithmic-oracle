# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-22

### Added
- 107 symbolic systems from diverse civilizations
- 27+ evolutionary optimization engines
- 27 analysis modules
- 153+ connected Python libraries
- GitHub repository with full CI/CD

### Symbolic Systems
- Astrology: Western, Vedic, Chinese, Korean, Egyptian, Mayan
- Divination: I-Ching, Tarot, Geomancy, Runes, Ogham
- Numerology: Pythagorean, Hebrew, Chinese, Roman
- Calendars: Gregorian, Jalali, Hijri, Mayan, Chinese
- Gematria: Hebrew, Greek, Agrippa, English Qaballa
- NLP: spaCy, Sentence Transformers, LangDetect

### Evolution Engines
- DEAP, PyMOO, Nevergrad, CMA-ES, Bayesian Optimization
- MealPy, NiaPy, Inspyred, NEAT, PyEasyGA
- And 17 more engines

### Analysis Modules
- Entropy: Shannon, Sample, Permutation
- Chaos: Lyapunov, DFA, Hurst Exponent
- Fractals: Box-counting, Fractal Dimension
- Causal: PC Algorithm, GES
- Bayesian: Network Learning, Inference

## [0.1.0] - 2026-06-17

### Added
- Initial project structure
- Core symbolic system framework
- Basic evolution engine
- Memory and persistence layer
